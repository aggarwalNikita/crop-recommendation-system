🎯 About This Project

This is a machine learning-based crop recommendation system that helps farmers make data-driven decisions.

✨ Features:

  • 🤖 Random Forest Classifier for predictions
  • 🌍 Support for 311 Indian districts
  • 📊 Real-time crop recommendations
  • 📈 Yield information and insights

🚀 Technologies Used:

  • Backend: Python, Flask
  • Machine Learning: scikit-learn
  • Data Processing: Pandas, NumPy
  • Frontend: HTML5, CSS3
